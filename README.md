# Freelancer Website

Bu proje, bir freelance web geliştiricisi ya da tasarımcısı için oluşturulmuş bir portföy sitesi.

## Özellikler

- Responsive tasarım
- Anasayfa, Hakkımda, Hizmetler, Projeler ve İletişim bölümleri
- İletişim formu (statik ya da basit backend ile entegre edilebilir)
- Google Fonts ve ikonlar

## Teknolojiler

- HTML  
- CSS  
- JavaScript
- SASS (SCSS)
